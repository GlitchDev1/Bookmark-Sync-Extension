Instructions

This site will guide you through the whole process of syncing your bookmarks via GitHub repository.

Setup GitHub Repository

  1. Create an account on GitHub
  2. Create a private repository on GitHub (maybe we can skip this part as extension is creating it)
  3. Create a personal access token with repository read & write and set expiration to none, otherwise the extension will stop working
  4. Copy the token to your clipboard, as it's the only time GitHub will display it

Configure Extension

  1. Open the settings
  2. Enter your GitHub username
  3. Paste in your access token
  4. Enter a repository name
Now you are good to go with this extension.

How to use

To sync your bookmarks, simply open the extension popup, then select all the bookmark folders you want to have synced.

From now on every bookmark that gets added into a synced folder will be automatically synced.

After that just click Sync and your bookmarks are uploaded. On a different device
